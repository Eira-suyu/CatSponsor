1. 引言

1.1 编写目的

本文档旨在明确“喵助”校园流浪猫TNR可视化助养平台的各项需求，为后续的系统设计、开发实现和验收测试提供依据。本文档面向项目开发者、指导教师及所有利益相关方。

1.2 适用范围

本文档适用于“喵助”校园流浪猫TNR可视化助养平台V1.0（MVP版本）的开发全过程，涵盖功能性需求、非功能性需求、数据需求、接口需求和界面需求等方面。

2. 项目概述

2.1 项目背景

校园流浪猫问题是全国高校普遍面临的社会问题。目前大多数高校的流浪猫管理工作依赖于学生社团自发组织，存在资金筹措困难、进度不透明、参与门槛高等问题。本项目旨在通过数字化手段，构建一个可视化、可交互、可传播的TNR助养平台，降低公众参与门槛，提高项目透明度，推动校园流浪猫问题的科学解决。

6.  数据实体定义

实体名称

实体描述

核心字段

猫咪(Cat)​

记录校园流浪猫的基本信息和医疗状况

id, name, gender, age, breed, status

助养记录(Donation)​

记录用户的每一次助养行为

id, catId, userId, amount, message

里程碑(Milestone)​

记录猫咪TNR过程中的关键节点

id, catId, type, title, date

用户(User)​

记录平台注册用户的信息

id, name, campus, level

消息(Message)​

记录系统通知和用户互动消息

id, userId, type, content, read

系统配置(System)​

记录平台的基础配置信息

id, key, value
